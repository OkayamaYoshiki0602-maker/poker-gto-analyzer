
### Rule #2: 相手のシグナル尊重

```
相手 1回 call → ⚠️ 警戒
相手 2回 call → 🚨 ブレーキ
相手 3回 call → 🛑 完全停止
```

### Rule #3: ポジション意識

```
Early (UTG-MP): Premium のみ (JJ+, AK)
Late (CO-BTN): 広く (77+, A9s+)
SB/BB: Defense、aggressive 禁止
```

### Rule #4: ポット管理

```
Pot 40bb 超えたら → 立ち止まる
Weak hand → Check-fold
Strong hand → Value 取る
```

### Rule #5: Pot Odds 計算

```
Call 前に: Call/(Call+Pot) vs Equity
Equity > Pot odds → Call
Equity < Pot odds → Fold
```

## 💡 Pot Odds クイックリファレンス

```
Flush draw (9 outs): 18% equity
→ Pot odds 20% 以下で call

OESD (8 outs): 16% equity
→ Pot odds 18% 以下で call

Gutshot (4 outs): 8% equity
→ Pot odds 10% 以下で call

Overcards (6 outs): 12% equity
→ Pot odds 14% 以下で call
```

**計算方法:**
```
Call amount / (Call + Pot) = Pot odds
例: 10bb call, pot 30bb
→ 10/(10+30) = 25%
```
