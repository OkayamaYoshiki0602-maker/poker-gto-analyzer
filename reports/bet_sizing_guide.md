## 📊 ストリート別ベットサイジング

### フロップ（Flop）

#### C-bet（継続ベット）
```
標準サイズ: 1/3 - 1/2 pot (33-50%)

小さいサイズ (1/3 pot) を使う時:
✅ ドライボード（連続性が低い）
✅ 相手が fold しやすい
✅ ポジションがある（IP）
例: K-7-2 rainbow

大きいサイズ (1/2 - 2/3 pot) を使う時:
✅ ウェットボード（ドローが多い）
✅ 相手のコール頻度が高い
✅ OOP（out of position）
例: Q-J-9 two-tone
```

#### バリューベット
```
標準サイズ: 1/2 - 2/3 pot (50-67%)

セットやツーペアなど:
✅ 2/3 pot が標準
✅ 相手のドローを守る
✅ pot を膨らませ始める
```

---

### ターン（Turn）

#### バリューベット
```
標準サイズ: 2/3 - 3/4 pot (67-75%)

強いハンド（セット以上）:
✅ 2/3 - 3/4 pot
✅ リバーでの all-in を見据えたサイジング
✅ ポットを大きく膨らませる
```

#### ブラフ
```
サイズ: 1/2 - 2/3 pot (50-67%)

ターンでのブラフ:
✅ バリューベットと同じサイズ
✅ バランスを取るため
✅ 相手に難しい判断を強いる

小さいブラフ (1/3 pot):
✅ プローブベット
✅ 相手が check した後
✅ 情報収集目的
```

---

### リバー（River）

#### バリューベット

##### スモールサイズ（1/3 - 1/2 pot）
```
使う状況:
✅ ミディアムストレングス（トップペアなど）
✅ 相手が fold しやすい
✅ 薄いバリュー（thin value）

例:
ボード: K-9-4-2-7
あなた: K-Q（トップペア）
→ 1/2 pot でバリュー
```

##### ミディアムサイズ（2/3 pot）
```
使う状況:
✅ 強いハンド（ツーペア、セット）
✅ 標準的なバリューベット
✅ 相手がコールする頻度が高い

例:
ボード: Q-J-8-5-2
あなた: Q-Q（セット）
→ 2/3 pot でバリュー
```

##### ラージサイズ（pot - オーバーポット）
```
使う状況:
✅ ナッツまたはナッツに近い
✅ 相手がトラップされている
✅ ポラライズドレンジ

例:
ボード: A-K-Q-J-T（ストレートボード）
あなた: K-J（フルハウス）
→ pot サイズでバリュー
```

#### ブラフ 🔥

**これが重要！**

##### 基本原則
```
🔥 ブラフサイズ = バリューベットサイズ

理由:
✅ バランスを取る
✅ 相手が判断できなくする
✅ GTO 的に正しい
```

##### スモールブラフ（1/3 - 1/2 pot）
```
❌ ほとんど使わない

例外的に使う状況:
- 相手が非常に fold しやすい（90%+）
- 小さなポットで争いたくない
- リスクを最小化したい

成功率必要: 25-33%
```

##### ミディアムブラフ（2/3 pot）
```
✅ 最も一般的なリバーブラフサイズ

使う状況:
✅ 標準的なブラフ
✅ 相手のレンジが広い
✅ ポットを取りたいが、リスクも管理

必要な成功率: 約 40%

例:
ボード: K-9-7-3-2 rainbow
あなた: A-Q（A-high）
相手が check → 2/3 pot ブラフ
```

##### ラージブラフ（pot - オーバーポット）
```
✅ ポラライズド戦略で使う

使う状況:
✅ 相手がトラップされている可能性
✅ ナッツブラフ（ブロッカー持ち）
✅ 相手のレンジが限定的

必要な成功率: 50-60%

例:
ボード: J♠T♠5♠2♣9♠（フラッシュ完成）
あなた: A♠7♠（ナッツフラッシュブロッカー）
相手 check → pot サイズブラフ

理由:
- あなたは A♠ 持ち = 相手はナッツを持てない
- 相手のレンジは K-high flush 以下
- 大きくブラフしてプレッシャー
```

## 🎓 ポットオッズとブラフサイズの関係

### 数学的分析

#### 1/3 pot ブラフ
```
ポットサイズ: 100bb
ベット: 33bb
新しいポット: 133bb

相手のポットオッズ: 33:133 = 25%
→ 相手は 25% の勝率で call できる

あなたの必要成功率: 33/(33+100) = 25%
→ 25% で成功すれば break even

結論: 相手が 75% fold する必要がある
→ 😕 リバーで 75% fold は難しい
→ ❌ リバーブラフには小さすぎる
```

#### 2/3 pot ブラフ
```
ポットサイズ: 100bb
ベット: 67bb
新しいポット: 167bb

相手のポットオッズ: 67:167 = 40%
→ 相手は 40% の勝率で call できる

あなたの必要成功率: 67/(67+100) = 40%
→ 40% で成功すれば break even

結論: 相手が 60% fold する必要がある
→ ✅ リバーで 60% fold は現実的
→ ✅ 標準的なブラフサイズ
```

#### pot サイズブラフ
```
ポットサイズ: 100bb
ベット: 100bb
新しいポット: 200bb

相手のポットオッズ: 100:200 = 50%
→ 相手は 50% の勝率で call できる

あなたの必要成功率: 100/(100+100) = 50%
→ 50% で成功すれば break even

結論: 相手が 50% fold する必要がある
→ ✅ ポラライズド状況で有効
→ ✅ ブロッカー持ちで使う
```

## 🎯 実践例

### 例1: リバーでのスタンダードブラフ

**状況:**
```
ポジション: BTN vs BB
ボード: K♥9♣7♠4♦2♠
ポット: 40bb

あなたのハンド: A♠Q♠（A-high）

BB check
→ あなたのアクション?
```

**分析:**
```
✅ ブラフに良い状況:
- 相手が check = weakness
- あなたは position がある
- ストーリーが合う（フロップから攻撃）

❌ 1/3 pot (13bb) ブラフ:
- 相手は 25% の勝率で call
- 弱いペアでも call される可能性

✅ 2/3 pot (27bb) ブラフ: 推奨
- 相手は 40% の勝率で必要
- 弱いペアは fold せざるを得ない
- 標準的なサイズ

❌ pot (40bb) ブラフ:
- リスクが高い
- この状況では過剰
```

**推奨アクション:**
```
Bet 27bb (2/3 pot)

期待値計算:
成功（60% の確率）: +40bb
失敗（40% の確率）: -27bb

EV = (0.6 × 40) - (0.4 × 27) = 24 - 10.8 = +13.2bb
→ 期待値プラス！
```

### 例2: リバーでのポラライズドブラフ

**状況:**
```
ポジション: CO vs BB
ボード: Q♠J♠T♠5♣2♠（フラッシュ完成）
ポット: 50bb

あなたのハンド: A♠9♦（A♠ ブロッカー）

相手の考えられるレンジ:
- K♠x♠（ナッツ）
- 小さなフラッシュ
- ペア（負けている）

BB check
→ あなたのアクション?
```

**分析:**
```
✅ 大きなブラフに良い状況:
- A♠ ブロッカー = 相手はナッツを持てない
- 相手が check = K♠ を持っていない可能性
- フラッシュボード = プレッシャーをかけられる

❌ 2/3 pot (33bb) ブラフ:
- 小さなフラッシュでも call されやすい
- プレッシャーが足りない

✅ pot (50bb) ブラフ: 推奨
- 相手に 50% の勝率を要求
- 小さなフラッシュでも fold を強制
- A♠ ブロッカーが機能

✅ オーバーポット (70bb) ブラフ: 上級
- さらに強いプレッシャー
- バランスが取れていれば有効
```

**推奨アクション:**
```
Bet 50bb (pot サイズ)

理由:
1. A♠ ブロッカーで相手のナッツを除外
2. pot サイズで小フラッシュを fold させる
3. ポラライズドレンジを表現
```

## 📊 まとめ: ベットサイジング早見表

### フロップ

| 目的 | サイズ | 状況 |
|------|--------|------|
| C-bet (ドライ) | 1/3 pot | 相手 fold しやすい |
| C-bet (ウェット) | 1/2 pot | ドロー多い |
| バリュー | 1/2 - 2/3 pot | セット等 |

### ターン

| 目的 | サイズ | 状況 |
|------|--------|------|
| バリュー | 2/3 - 3/4 pot | セット以上 |
| ブラフ | 1/2 - 2/3 pot | バリューと同じ |
| プローブ | 1/3 pot | 情報収集 |

### リバー（重要！）

| 目的 | サイズ | 状況 |
|------|--------|------|
| 薄いバリュー | 1/3 - 1/2 pot | トップペア等 |
| 標準バリュー | 2/3 pot | ツーペア、セット |
| 強いバリュー | pot+ | ナッツに近い |
| **標準ブラフ** | **2/3 pot** | **一般的なブラフ** |
| **大きなブラフ** | **pot - オーバーpot** | **ポラライズド、ブロッカー** |
| ❌ 小さなブラフ | ❌ 1/3 pot | ❌ ほぼ使わない |

## 💡 重要な原則

### 1. バランスの法則
```
🔥 ブラフサイズ = バリューベットサイズ

なぜ?
→ 相手が判断できなくなる
→ GTO 的にバランスが取れる
→ exploited されにくい
```

### 2. リバーブラフは大きく
```
❌ 1/3 pot ブラフ: 相手が 25% で call → 簡単
✅ 2/3 pot ブラフ: 相手が 40% で call → 難しい
✅ pot ブラフ: 相手が 50% で call → 非常に難しい
```

### 3. 状況に応じた調整
```
ドライボード: 小さめ (1/3 - 1/2 pot)
ウェットボード: 大きめ (2/3 - pot)

相手 fold しやすい: 小さめ
相手 call 頻度高い: 大きめ

ブロッカーあり: 大きめ
ブロッカーなし: 標準
```
